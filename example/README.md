# RDuuke\\Router example

1. Install composer

      ```text
      $ cd Router
      $ composer install
      ```

2. Run php server

      ```text
      $ cd Router
      $ php -S localhost:8080 -t example example/index.php
      ```

3. Open browser

      Open http://localhost:8080 in your browser and you will see 'Welcome RDuuke\\Router!'
